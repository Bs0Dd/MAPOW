# MAPOW (Mobile Action POWer) 

Mobile Action USB cable charging control.

![Cables](/macables.jpg?raw=true)

Works for old Mobile Action USB data cables (**MA-xxxxP**) based on Prolific **PL2303** USB-TTL chip.  
Enabling and disabling charge function. Better than using original Windows-only app (custom MA driver required).  
Just **set** COM or ttyUSB number and **use**!

Working on **Linux** and **Windows (95+)**.

![Linux](/maplin.png?raw=true)
![Windows](/mapwin.png?raw=true)


Sources for **GCC** (Linux) and **Pelles C 7.0** (Windows).
